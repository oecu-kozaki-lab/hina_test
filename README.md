# 評価手順
https://oecu-kozaki-lab.github.io/hina_test/

## 1. プレイ実行
各データセット（A〜E）について、以下の手順でプレイを行ってください。

1. **データセットを選択**
   - スタート画面でデータセット（例：A:閲覧数12000以上）を選択します。

2. **両方のモードでプレイ**
   - **「ふりがなあり」** で2回以上プレイします。
   - **「ふりがななし」** で2回以上プレイします。
   - ※1回プレイするごとに5問出題されます。

## 2. プレイ中の評価
各問題を解いた後、結果画面で以下のいずれかのフィードバックボタンを押してください。

- **💡 スッキリした（閃いた！）**
  - 理屈がわかって納得できた場合。
- **📚 スッキリしなかった（知識問題）**
  - 知らないと解けない、または納得感が薄い場合。
-  正解に納得いかない場合はその内容をフィードバック欄に書いてください      

## 3. データの保存とリセット
1つのデータセット（例：データセットA）について、「ふりがなあり」「なし」の両方を規定回数（各2回以上）プレイし終えたら、以下の手順を行います。

1. **評価レポートを表示**
   - スタート画面または結果画面の「📊 評価レポート」ボタンを押します。

2. **スクリーンショットを保存**
   - 表示された評価レポート画面のスクリーンショットを撮ります。

3. **データをダウンロード**
   - 「📥 データをCSVでダウンロード」ボタンを押し、CSVファイルを保存します。
   -  ファイル名は「選んだデータセット_ふりがな有り無し_学年」のようにしてください。（例：A_有り_4年）

4. **評価をリセット**
   - 「🗑️ 評価をリセット」ボタンを押し、これまでの記録を消去します。

---
**上記の手順（1〜4）を、すべてのデータセット（A〜E）に対して繰り返してください。**
