# 評価手順
謎解き→https://oecu-kozaki-lab.github.io/hina_test/

## 1. プレイ実行
各データセット（A〜C）について、以下の手順でプレイを行ってください。

1. **データセットを選択**
   - スタート画面でデータセット（例：A:閲覧数12000以上）を選択します。  
　　※デフォルトでデータセットはAになっているのでB,Cやる際はデータセットしっかり確認してから始めてください。ホームに戻るとデータはAになっています。

2. **両方のモードでプレイ**
   - **「ふりがなあり」** で1回以上プレイします。
   - **「ふりがななし」** で1回以上プレイします。
   - ※1回プレイするごとに5問出題されます。

## 2. プレイ中の評価
各問題を解いた後、結果画面で5段階評価してください。

*5段階評価
  - 知らないと解けない、または納得感が薄い場合。
-  正解に納得いかない場合、間違いがある場合はその内容をフィードバック欄に書いてください      

## 3. データの保存とリセット
1つのデータセット（例：データセットA）について、「ふりがなあり」「なし」の両方を規定回数（各1回以上）プレイし終えたら、以下の手順を行います。

1. **評価レポートを表示**
   - 結果画面の「📊 評価レポート」ボタンを押して。

2. **データをダウンロード**
   - 「📥 データをCSVでダウンロード」ボタンを押し、CSVファイルを保存します。
   -  ファイル名は「選んだデータセット_学年_名前」のようにしてください。（例：A_4年_中川）

3. **評価をリセット**
   - 「🗑️ 評価をリセット」ボタンを押し、これまでの記録を消去します。

4. **ダウンロードした物の提案手法**
   - Googleドライブに提出用のフォルダーを作成するのでそのファイル内に自分の名前のフォルダを作成してダウンロードした物をいれてください。
   -提出場所：https://drive.google.com/drive/u/0/folders/1Uu8fjEmx3Zx9weW3iWdtb0CCjXrVPiw9  

**上記の手順（1〜4）を、すべてのデータセット（A〜C）に対して繰り返してください。**
